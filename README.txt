1. Скачайте и установите VS Code (если не установлен):
https://code.visualstudio.com/

2. Установите расширение Live Server:
   2.1. Откройте VS Code
   2.2. Перейдите в раздел расширений (Ctrl+Shift+X)
   2.3. Найдите Live Server (автор: Ritwick Dey) и установите

3. Запустите проект:
   3.1. Откройте папку проекта в VS Code
   3.2. Найдите файл index.html в проводнике
   3.3. Правой кнопкой мыши → Open with Live Server

  Проект откроется в браузере по адресу http://127.0.0.1:5500/
  Важно! Браузер должен поддерживать WebGPU
